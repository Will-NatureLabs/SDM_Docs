del_time_dimension(df)[source]

Deletes the temporal dimension of a DataFrame by removing the ‘Day’ column and reordering NaNs within each ‘site’ group.

The purpose of this is to handle the data collection technique of rotating sensors around the study area. This technique results in patchy data spatially and temporally.

Dropping the time dimension essentially condenses the data into a single survey, containing recordings over space while disregarding when the recordings were made.

This function takes a DataFrame with a ‘site’ column, a ‘Day’ column (time dimension), and one or more additional columns (presumed to be species data or similar). The function performs the following operations:

  1. Identifies columns that are neither ‘site’ nor ‘Day’.

  2. Sorts each of these columns within each ‘site’ group to place NaNs at the end, while preserving the relative order of other values.

  3. Removes rows where all of the identified columns have NaN values.

  4. Removes rows where all columns (including ‘site’) are NaN.

  5. Deletes the ‘Day’ column.

Parameters:

df (pandas.DataFrame) – The input DataFrame containing ‘site’, ‘Day’, and other relevant columns.

Returns:

A DataFrame with the time dimension removed and NaNs reorganized.

Return type:

pandas.DataFrame

preprocess_y_data(df=None, folder_path=None, species_list=None, missing_threshold=0, balance_threshold=20, del_time=True, verbose=True)[source]

Preprocesses CSV data files containing species observation data based on the folder they reside in and returns a single processed DataFrame.

This method is designed to handle specific CSV data files located within folders. The preprocessing applied to the data includes applying missing data thresholds, balance thresholds, and optionally deleting the time dimension.

Parameters:

  • folder_path (str, optional) – Path to the directory where the data folders are located. If not provided, the method retrieves a default path stored within the class.

  • species_list (list, optional) – A list of species names to be included in the processed DataFrame. If not provided, all species found in the CSV files will be included.

  • missing_threshold (int, optional) – Percentage threshold to filter out columns with missing data greater than this threshold. Defaults to 0.

  • balance_threshold (int, optional) – Threshold to filter out columns based on the balance score. Defaults to 20.

  • del_time (bool, optional) – Flag indicating whether to delete the time dimension. Defaults to True.

  • verbose (bool, optional) – Flag indicating whether to print informational messages during processing. Defaults to True.

Returns:

A processed DataFrame containing the concatenated and cleaned data from all the input CSV files.

Return type:

pd.DataFrame

Key Functionalities:

  • Reads CSV files from a given directory and applies specific preprocessing such as missing data and balance thresholds.

  • Optionally removes the time dimension if del_time is True.

  • Aggregates and concatenates all the processed data.

  • Can filter data by species if a species_list is provided.

Notes

  • Removes data with fewer examples than the balance_threshold to prevent errors when trying to split the data into train/validation/test datasets.

Examples

>>> preprocess_y_data(missing_threshold=40, balance_threshold=40, folder_path='./data/folders')

Uses XGBoost to predict and fill NA values for specific columns based on coordinates.

This function leverages the power of the XGBoost algorithm to make predictions for the columns in the DataFrame containing NA values. The primary predictors for these columns are coordinates.

Parameters: - df (pd.DataFrame): The input dataframe that contains the data with potential NA values. - lon_col (str): The name of the column that represents the longitude. Defaults to ‘lon’. - lat_col (str): The name of the column that represents the latitude. Returns: - pd.DataFrame: A DataFrame with NA values predicted and filled using XGBoost.

Note

fill_na_with_avg(df, pattern='isPresent')[source]

Fills NA values for specific columns in a DataFrame with the average of their respective site’s values.

This function replaces NA values in columns containing the ‘isPresent’ pattern with the average of non-NA values for that column within each site. This operation is done in a site-specific manner and does not affect sites that have no records for a species.

Parameters: - df (pd.DataFrame): The DataFrame containing the data with potential NA values.

Returns: - pd.DataFrame: A DataFrame with NA values replaced by the site-specific average for columns with the ‘isPresent’ pattern.

Note

Only columns that contain the pattern and are of a numeric datatype are processed. Ensure that the DataFrame has the ‘site’ column available as it is used for grouping data. Does not affect sites with no records for a species
