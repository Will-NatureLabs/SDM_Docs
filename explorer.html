create_species_map(species=None, time_column='Day', lon_column='lon', lat_column='lat', summarize=False, endure=True, verbose=True, pattern='_isPresent', download=False)[source]

Creates a map visualization of the presence of different species over time. Gives insight into the survey pattern and the recordings of the raw data.

Parameters:

  • species (list, optional) – List of species to filter by, case insensitive and returns partial matches. If None, all species are included.

  • time_column (str, optional) – Column name in DataFrame representing time. Defaults to ‘Day’.

  • lon_column (str, optional) – Column name in DataFrame representing longitude. Defaults to ‘lon’.

  • lat_column (str, optional) – Column name in DataFrame representing latitude. Defaults to ‘lat’.

  • summarize (bool, optional) – If True, summarizes the species presence data before plotting. Defaults to False.

  • endure (bool, optional) – If True, no duration is set for the animation. If False, a duration of ‘P1D’ is set. Defaults to True.

  • verbose (bool, optional) – If True, prints debug information. Defaults to True.

  • pattern (str, optional) – String pattern used to identify columns containing species presence information. Defaults to ‘_isPresent’.

  • download (bool, optional) – If True, the generated map will be downloaded as an HTML file. Defaults to False.

Returns:

A Folium map object with the presence of different species visualized.

Return type:

folium.Map

Example

>>> create_species_map(species=['Eleutherodactylus_coqui', 'Margarops_fuscatus'], download=True)
