Fetches and aggregates a collection of images from Google Earth Engine over a given date range, spatially filters the collection to the specified geometry, and applies an aggregation method. Defaults to using the past 2 years of data if start_date and end_date are not provided.

Parameters:

  • collection_id (str) ‚Äì The ID of the Earth Engine image collection to use.

  • start_date (str, optional) ‚Äì The start date for the image collection filter (inclusive), formatted as ‚ÄòYYYY-MM-DD‚Äô. Defaults to two years before the current date if not provided.

  • end_date (str, optional) ‚Äì The end date for the image collection filter (inclusive), formatted as ‚ÄòYYYY-MM-DD‚Äô. Defaults to the current date if not provided.

  • filter_geometry (geopandas.GeoDataFrame, optional) ‚Äì A GeoDataFrame specifying the spatial extent to filter the collection. Defaults to None.

  • aggregation_method (str, optional) ‚Äì The method used to aggregate images over time. Defaults to ‚Äòmean‚Äô. Supported methods: ‚Äòmean‚Äô, ‚Äòmedian‚Äô, ‚Äòmin‚Äô, ‚Äòmax‚Äô, ‚Äòsum‚Äô, ‚Äòmosaic‚Äô.

Returns:

An Earth Engine image object resulting from the aggregated collection.

Return type:

ee.Image
