main_module

class species_distribution_modeling.main_module.SpeciesDistributionModeling(base_folder)[source]

This class acts as an entry point to the species_distribution_modeling module. Upon initialisation, it creates a new project, setting up the folder structure inside of the base_folder.

It also creates instances of each submodule and passes utils to each of the three main modules. This grants them easy access to the helper functions inside utils through their self.utils attribute.

The three core modules:

  1. Explorer

  2. Processing

  3. Modeling

Finally, it assigns the paths of each of the project’s folders to self attributes, allowing each module to use predetermined paths as defaults to streamline workflows.
